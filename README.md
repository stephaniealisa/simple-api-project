#simple-api-project

## Overview
A command-line Python application for managing a simple to-do list. Users can add, view, and delete tasks through a menu-driven interface.

## Features
- Add tasks
- View all tasks
- Delete tasks by number
- Handles invalid input gracefully
- Demonstrates Python functions, lists, and user input
- Includes automated tests using pytest

## Technologies Used
- Python
